PHP / ISO 8601 type formatted dates

Date Axes also provide powerful formatting features. This allows custom formatter strings to be used to format axis tick labels precisely the way you want.

Bellow we have a time series in ISO 8601 the form - this form is widely used in database applications, but also much used in PHP. Because the format is YYYY-MM-DD, it's very easy to sort.

Week numbers are very popular in Europe, and on the X-axis we print out the year and the week number.

If you click on the legned, you'll notice that you can togle it on or off. This is very usefull when there is a lot of data presented.
