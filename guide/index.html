Getting started

  • Insert javascript dependencies replacing <jqplot_dist_directory> by the directory you've installed the distributions files.

  • Insert an html chart container into <body> html.

  • Insert the code to generate the chart using the element id defined.

    $(function(){ // Chart datas var line1 = [[1,4], [2,6.5], [3,5.7], [4,9], [5,8.2]]; // Chart building system call. $.jqplot('chart', [line1]); });
